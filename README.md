# Проект Online_store

## Описание:

- в модуле creat_class  реализованы классы Category (название, описание, товары) 
  и Product (название, описание, цена, кол-во в наличии)


## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/meirabeloritskaya/Online_store.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
